# BibliotecaRosa

## Overview

This project is a simple library management system where I worked with basic data handling and application logic. The idea was to simulate how a system could organize and manage a collection of books, making it easier to store, search, and update information.

Even though it is a smaller project, it helped me strengthen my programming fundamentals and understand how to structure an application.

---

## How it works

The system allows you to manage a collection of books by storing relevant information such as titles, authors, and other details.

Through the application logic, users can add new books, search for existing ones, and organize the data in a structured way. The focus of this project was not only functionality, but also writing clean and understandable code.

---

## Technologies used

This project was developed using Python, focusing on core programming concepts such as data structures, functions, and basic data management.
